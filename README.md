# CloudPi User Guide Documentation

This directory contains the complete CloudPi user guide documentation in Markdown format, ready for MkDocs deployment.

## Structure

- `docs/` - All markdown documentation files
- `mkdocs.yml` - MkDocs configuration file

## Local Preview

To preview locally:

```bash
mkdocs serve
```

Then visit http://localhost:8000

## Deployment

This documentation is automatically deployed to GitHub Pages at:
https://purpledatainc-tx.github.io/mkdocs/

## Contributing

When adding new pages:
1. Create markdown file in appropriate directory
2. Update `mkdocs.yml` navigation
3. Test locally with `mkdocs serve`
4. Commit and push to trigger deployment
