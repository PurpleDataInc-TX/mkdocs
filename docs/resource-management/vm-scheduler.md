# Virtual Machine Scheduler

## Overview

The **Virtual Machine Scheduler** feature offers powerful automation for managing start and stop operations of virtual machines. Organizations can optimize resource utilization, reduce cloud costs, and ensure efficient cloud infrastructure management.

## Key Features

### Comprehensive Schedule Management

- **Automated Operations:** Enable automatic start/stop during specified times
- **Vacation and Custom Schedules:** Create schedules for holidays, weekends, specific needs
- **Multi-Region Support:** Manage schedules across different cloud regions

### Centralized Schedule Overview

- **Detailed Schedule View:** Access all active schedules
- **Actionable Controls:** Modify, delete, or manually override schedules
- **Execution Logs:** Monitor activities and statuses

### Data-Driven Recommendations

- **Optimized Suggestions:** Receive schedule recommendations based on usage patterns
- **Streamlined Implementation:** Apply recommendations with minimal configuration

## Setup and Configuration

### Accessing the Scheduler

2. Navigate to **Automation** â†’ **Scheduler**
3. Explore three main screens:
   - **Schedules:** Manage and review active schedules
   - **Recommendations:** View usage-based suggestions
   - **Resources:** Identify eligible resources

## Navigating Schedules

### Schedules Screen

View list of schedules with:
- Schedule name
- Type (Snooze, Start, Stop)
- Description
- Associated resources

### Available Actions

**View:** Examine schedule details without changes

**Manual Override:** Temporarily pause/activate schedule for individual instances

**Edit:** Modify schedule configuration

**Schedule Logs:** View execution history

**Delete:** Remove schedule permanently

**Export Instance Details:** Download spreadsheet of all schedules

## Creating Schedules

### From Recommendations

1. Navigate to Recommendations screen
2. Review suggested snooze schedules
3. Select recommendations
4. Click **Create Schedule**
5. Customize and save

### From Resources

1. Navigate to Resources screen
2. Select one or more resources
3. Click **Create Schedule**
4. Define:
   - Schedule Type (Vacation or Custom)
   - Description
   - Start/End Date
   - Stop/Start Times
   - Time Zone
5. Save schedule

## Best Practices

1. **Start Conservative:**
   - Begin with non-production environments
   - Test schedules thoroughly
   - Gradually expand to production

2. **Use Recommendations:**
   - Review usage analysis
   - Implement high-confidence suggestions
   - Track savings

3. **Regular Reviews:**
   - Quarterly schedule audits
   - Adjust for changing patterns
   - Remove obsolete schedules
