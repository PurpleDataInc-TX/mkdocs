# Resource Scheduling

## Overview

The **Resources** section allows viewing and managing resources eligible for scheduling actions.

## Key Features

- Select resources for scheduling
- Create custom schedules
- View resource details
- Tag-based filtering
- Region filtering
- Export resource data

## Managing Resources

### Selecting Resources

1. Navigate to Resources page
2. Check boxes for desired resources
3. Click **Create Schedule** button

### Creating Schedule

Configure:
- Schedule Type (Vacation or Custom)
- Description
- Start/End Date  
- Stop/Start Times
- Time Zone

### Viewing Resource Details

Use **View** action to see:
- Resource ID
- Region
- Tags
- Current state
- Associated schedules
