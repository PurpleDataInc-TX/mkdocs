## Common Journeys

### Onboard a New Cloud Provider

1. Open [Getting Started](GettingStarted.md)
2. Review [Provider Connections](ProviderConnections.md)
3. Follow [Connect AWS](ConnectAWS.md), [Connect Azure](ConnectAzure.md), or [Connect GCP](ConnectGCP.md)
4. Validate ingestion in [Billing Hub](MultiCloudBillingHub.md)

### Roll Out Governance and Automation

1. Review [Automation Overview](AutomationOverview.md)
2. Configure [Policies & Workflows](AutomationPolicies.md)
3. Set up [Policy YAML Sync](PolicyYAMLSync.md) if your team uses it
4. Review [Scheduler](Scheduler.md) and [VM Scheduler](AutomationVMScheduler.md)

### Configure Access and Security

1. Review [RBAC](rbac.md)
2. Configure [User Management](UserManagement.md)
3. Set up [Service Accounts](ServiceAccounts.md)
4. Review [Security](security.md) and [Audit Logging](AuditLogging.md)
