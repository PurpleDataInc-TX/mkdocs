# Auto Invoicing Policy

## Overview

**Auto Invoicing** is a built-in policy that automates invoice generation every month based on data usage, ensuring timely billing and reducing manual intervention.

## How to Set Up Time-Based Workflow

### Configuration Steps

1. **Workflow Name:** auto-invoicing

2. **Description:** Automatically generates invoices based on usage every month

3. **Schedule:**
   - Start Date: When workflow begins
   - Frequency: Monthly

4. **Billing Details:**
   - Billing View: Select view with usage data
   - Customer: Choose customer to bill
   - Add multiple billing configurations if needed

5. **Actions:**
   - Select: "Schedule Auto Invoicing"
   - Add additional actions if needed

6. **Save Workflow:** Click **Create Workflow** to activate

## Benefits

- Automated monthly invoice generation
- Consistent billing cycles
- Reduced manual workload
- Improved efficiency
- Decreased human error risk
