# Cost and Usage Types

## Overview
CloudPi supports multiple cost views to align with finance and engineering reporting needs. Use the cost type that matches your reporting standard.

## Common Cost Types
- **Billed Cost**: The invoiced amount from the cloud provider.
- **Effective Cost**: Cost after discounts, credits, or savings plan amortization.
- **List Cost**: The on-demand list price before discounts.

## Where Cost Types Are Used
- **Billing Hub** filters and charts
- **Dashboards** and saved views
- **Budgets and Forecasts**

## Best Practices
- Use **Billed Cost** for accounting and chargeback.
- Use **Effective Cost** for optimization and savings analysis.
- Use **List Cost** for capacity planning and pricing comparisons.
