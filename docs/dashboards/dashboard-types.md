# Dashboard Types

## Available Dashboard Categories

CloudPi provides multiple dashboard types tailored to different user roles and analysis needs. Each dashboard type presents data relevant to specific business functions.

## Executive Dashboard

**Purpose:** High-level strategic insights for leadership

**Target Audience:**
- C-level executives
- VP of Engineering/Infrastructure  
- CFO and Finance Leadership
- Business Unit Heads

**Key Metrics:**
- Total Cloud Spend
- Month-over-Month Change
- Year-over-Year Comparison
- Budget vs Actual
- Forecast Accuracy
- Cost per Business Unit
- Major Cost Drivers

**Visualizations:**
- Financial trend lines
- Budget burn rate
- Spend by department/project
- Cloud provider comparison
- Cost optimization savings realized

**Use Cases:**
- Board presentations
- Monthly business reviews
- Budget planning discussions
- Investment decisions

## Custom Dashboards

**Purpose:** User-defined views for specific needs

**Creation:** 
Workspace Admins and Users can create custom dashboards by:
1. Selecting data sources
2. Choosing visualizations
3. Configuring filters
4. Saving custom views

**Sharing:**
- Share with team members
- Set as default for projects
- Export for presentations
- Schedule automated reports

## Selecting the Right Dashboard

### Decision Matrix

| Your Role | Primary Goal | Recommended Dashboard |
|-----------|-------------|----------------------|
| Executive | Strategic oversight | Executive |
| Finance | Cost management | FinOps |
| Engineer | Performance/optimization | Technical |
| Analyst | Deep cost analysis | Cost Analysis |
| Manager | Budget tracking | Budget |
| Mixed needs | Flexible analysis | Custom |

### Dashboard Navigation Tips

1. **Start with Executive** - Get the big picture
2. **Drill into FinOps** - Understand cost details
3. **Switch to Technical** - Investigate resources
4. **Use Cost Analysis** - Deep-dive specific issues
5. **Monitor with Budget** - Track financial targets

## Dashboard Customization

### Available Customizations

All dashboard types support:
- **Filter Configuration** - Save preferred filters
- **Date Range Presets** - Quick time period selection
- **Metric Selection** - Choose which KPIs to display
- **Layout Adjustment** - Rearrange dashboard components
- **Export Options** - PDF, Excel, CSV formats

### Saving Custom Views

1. Configure dashboard with desired settings
2. Click "Save View" button
3. Name your custom view
4. Choose sharing options:
   - Private (only you)
   - Team (project members)
   - Organization (all workspace users)
5. Set as default (optional)

## Dashboard Refresh Rates

| Dashboard Type | Refresh Frequency | Last Updated Display |
|---------------|-------------------|----------------------|
| Executive | Every 6 hours | Yes |
| FinOps | Every 4 hours | Yes |
| Technical | Real-time* | Yes |
| Cost Analysis | Every 4 hours | Yes |
| Budget | Every 2 hours | Yes |

*Real-time for performance metrics; cost data updates every 4 hours

## Best Practices by Dashboard Type

### Executive Dashboard
- Review weekly or monthly
- Focus on trends, not details
- Use for strategic discussions
- Export for board presentations

### FinOps Dashboard
- Monitor daily
- Investigate anomalies immediately
- Track optimization progress
- Use for monthly financial reviews

### Technical Dashboard
- Check daily for alerts
- Monitor resource utilization
- Track compliance status
- Use for capacity planning

### Cost Analysis Dashboard
- Use for detailed investigations
- Compare time periods
- Analyze by multiple dimensions
- Export data for offline analysis

### Budget Dashboard
- Review weekly minimum
- Watch threshold alerts
- Update forecasts monthly
- Share with budget owners
