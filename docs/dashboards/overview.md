# Dashboard Overview

## Introduction

The **CloudPi Dashboard** serves as a unified platform for monitoring and managing your cloud environment. It provides rich visualizations and interactive reports that give teams a clear understanding of **costs**, **resources**, and **project performance** across multiple cloud providers.

The dashboard is role-based and project-awareâ€”ensuring every user views insights that are **relevant, actionable, and aligned** with their responsibilities.

## Key Features of the Dashboard

### Centralized Cloud Insights

The CloudPi Dashboard aggregates data across all projects and provides a single interface for:

- **Cost Reports** - Track spending across cloud providers
- **Budget Tracking** - Monitor budget consumption
- **Show back Reports** - Allocate costs to business units  
- **Executive Reports** - High-level summaries for leadership

After choosing the dashboard type, select a specific dashboard name from a list tailored to your project needs.

### Customizable Data Visualization

CloudPi dashboards offer flexible data views through multiple filter options:

#### Filters Available

- **Date Range** - Choose time periods (e.g., Last 90 Days, Month to Date, Year to Date)
- **Granularity** - Switch between Daily, Monthly, or Quarterly views
- **Cloud Provider** - Filter by AWS, Azure, GCP, or view combined data
- **Projects** - Focus on selected projects or project groups

This allows you to focus on particular areas of interest within your cloud environment.

## Dashboard Access by Role

### Workspace Admin
- Full access to all dashboards
- Can create custom organizational dashboards
- Access to cross-project analytics
- Executive summary views

### Workspace User
- Access to multi-project dashboards
- Executive level reports
- Customized reports across assigned projects
- Budget and forecast summaries

### Project Admin
- Full access to project-specific dashboards
- Team performance metrics
- Project cost analytics
- Resource utilization reports

### Project User
- Read-only access to project dashboards
- Personal task and assignment views
- Limited cost visibility (project scope only)

## Navigating the Dashboard

### Accessing the Dashboard

1. Log in to your CloudPi account
2. From the left navigation menu, click on the **Dashboard** icon
3. The main Dashboard page opens, displaying an overview panel with filters and visual reports

### Selecting Dashboard Types

At the top of the dashboard, you'll find multiple dropdown filters that control what type of data is displayed.

Use the **Dashboard Type** dropdown to choose the desired view:
- **Executive** - High-level KPIs and summaries
- **FinOps** - Financial operations metrics
- **Technical** - Resource and performance data
- **Cost Analysis** - Detailed spending breakdowns
- **Budget** - Budget tracking and forecasts

Dashboard options are dynamically customized based on:
- Your selected project(s)
- Your user role (e.g., Admin, Executive, FinOps Analyst)

This ensures that each user views insights that are relevant to their scope and responsibilities.

### Choosing Specific Reports

After selecting the Dashboard Type, additional dropdowns appear for further customization:

1. In the **Report Selector**, choose the specific report you want to view:
   - Cost by Projects
   - Budget Overview
   - Resource Utilization
   - Service Breakdown
   - Tag Analysis
   - Regional Spend

2. You can also select:
   - **Date Range** - Define the analysis period
   - **Granularity** - Set time intervals
   - **Cloud Provider** - Filter by provider
   - **Projects** - Select specific projects

The dashboard then updates automatically, showing relevant charts and tables for your selection.

### Customizing Data Views

Refine and personalize your dashboard data using filters located across the top:

- **Date Range Filters** - Analyze cost and usage trends across specific time frames
- **Granularity Filters** - View your data at daily, monthly, or quarterly levels for trend analysis
- **Cloud Provider Filters** - Narrow your view to a particular cloud provider or compare across all
- **Project and Dimension Filters** - Focus on specific projects, services, or custom tags

## Best Practices

### Effective Dashboard Usage

1. **Start Broad, Then Narrow**
   - Begin with executive summary
   - Identify areas of interest
   - Drill down into specifics

2. **Use Saved Views**
   - Create custom dashboard configurations
   - Save commonly used filter sets
   - Share views with team members

3. **Regular Monitoring**
   - Check dashboards daily/weekly
   - Set up alert thresholds
   - Review trends regularly

4. **Combine Multiple Views**
   - Compare time periods
   - Analyze by different dimensions
   - Cross-reference with budget data

### Performance Tips

- **Limit Date Ranges** - Shorter periods load faster
- **Use Granularity Wisely** - Daily data for short periods, monthly for long periods
- **Filter Projects** - Select only relevant projects
- **Cache Reports** - Frequently used reports are cached for faster access

## Troubleshooting

### Common Issues

**Dashboard Not Loading:**
- Check internet connection
- Verify project access permissions
- Clear browser cache
- Try different browser

**Data Not Updating:**
- Check last refresh time
- Billing data updates daily (usually by 9 AM)
- Click refresh button
- Verify data collection is active

**Charts Not Displaying:**
- Verify data exists for selected filters
- Check date range isn't too far in past
- Ensure project has billing data
- Try different granularity

**Slow Performance:**
- Reduce date range
- Filter to fewer projects
- Use monthly vs daily granularity
- Contact support if persistent
