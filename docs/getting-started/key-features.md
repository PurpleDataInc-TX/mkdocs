# Key Features

CloudPi offers comprehensive features for multi-cloud cost management:

## Projects & Groups Management
Organize your cloud resources with project and project group hierarchy.

## Role-Based Access Control (RBAC)
Securely manage who has access to what with granular permissions.

## User Management
Add or remove users, assign roles, and manage permissions across workspaces and projects.

## Team Collaboration
- Task Management
- Ticketing Integration (Jira, ServiceNow, ADO)
- Notifications and Alerts
- Sharing and downloading capabilities

## Multi-Cloud Billing Hub
Centralize your billing for chargeback, show back management, and custom invoice generation across AWS, Azure, and GCP.

## Budget Management
Set and track budgets at project and service group levels with automated alerts.

## Cost Optimizations & Recommendations
Receive AI-driven suggestions to reduce expenses based on usage patterns.

## Workflow Integration & Automation
Streamline operations with customizable workflows and automated actions.

## Custom Dashboards
Generate reports based on tags, accounts, and custom filters for detailed insights.

## Tag Management
Implement tagging strategies for projects to enforce responsibility and compliance policies.

## VM Scheduler
Automated start/stop scheduling for virtual machines to optimize costs.

## Multi-Tenancy
Manage multiple clients or Business Units within a single CloudPi instance.

## Security Features
- Read-only cloud access
- Data encryption at rest and in transit
- SSO integration (Azure AD, OIDC)
- Audit logging
- Compliance reporting
